## Selected Projects

### 1. Wet Sensor-Based Menstruation Blood Detection System (Autism Support)

**Problem Statement**  
Autistic individuals may face difficulty in communicating discomfort during menstruation, leading to delayed care and health risks.

**Why This Matters**  
Early detection improves hygiene management, caregiver response time, and overall well-being for dependent users.

**Approach**  
- Designed a sensor-based detection system using resistance variation  
- Integrated ESP8266 for data handling and alert logic  
- Implemented threshold-based decision rules for reliable detection  

**Key Decisions Made**  
- Chose resistance-based sensing for cost-effectiveness and simplicity  
- Prioritized reliability and caregiver alerts over complex modeling  

**Outcome**  
- Achieved consistent detection under controlled conditions  
- Designed with scalability and real-world usability in mind  

**Tech Stack**  
ESP8266, Wet Sensor, Embedded Logic, Python (analysis)

### 4. IoT-Based Emergency Locator System (LoRa + GPS)

**Problem Statement**  
Emergency communication is unreliable in remote or network-deficient areas.

**Why This Matters**  
Reliable location sharing can be critical in high-risk or isolated environments.

**Approach**  
- Designed a communication system using ESP32, LoRa, and GPS modules  
- Enabled long-range encrypted message transmission  
- Displayed real-time system status using an LCD interface  

**Key Decisions Made**  
- Selected LoRa for long-range, low-power communication  
- Prioritized robustness and independence from cellular networks  

**Outcome**  
- Successfully transmitted location data over long distances  
- Demonstrated strong integration of hardware and communication logic  

**Tech Stack**  
ESP32, SX1276 LoRa, GPS Module, Embedded C, System Integration
